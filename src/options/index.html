Notion 저장 (선택)

답변을 💾 Notion 저장 버튼으로 지정한 데이터베이스에 바로 저장합니다. (제목 = 물어본 단어, 본문 = 답변 블록)

  1. notion.so/my-integrations 에서 internal integration 생성 → 토큰 복사
  2. 대상 DB 페이지 우상단 ⋯ → "연결(Connections)"에서 그 integration 추가
  3. 아래 DB ID / URL 칸에 DB 페이지 주소(URL)를 통째로 붙여넣기 (자동으로 ID만 추출 · URL 속 32자리 hex만 입력해도 됨)

사전 프롬프트

AI에게 답변 형식을 지시하는 system 프롬프트. 입력한 단어·질문은 이 지시에 따라 답변됩니다.

팝업 단축키는 chrome://extensions/shortcuts 에서 바꿀 수 있습니다 (기본 Alt+Q). 모든 변경은 자동 저장됩니다.
